# handoff 会话交接

目标：把当前会话压成一份交接文档，让下一个 session 的 agent 不用回溯对话就能无缝接手。

## 做法
- 交代清楚：目标、已达成的结论、当前进度、下一步该做什么。
- 已有产物——计划、笔记、commit、文件——用路径或链接引用，不把内容抄进来。
- 脱敏：密钥、个人信息不写入文档。
- 留一节"建议 skill"，点明下一个 session 起步该先用哪些 skill。
- 文档存到系统临时目录，不落在项目里。

## 裁剪
用户以参数说明了下个 session 的用途时，按该用途裁剪内容，只留相关部分。
